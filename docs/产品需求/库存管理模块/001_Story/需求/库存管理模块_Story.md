# 库存管理模块 - 用户故事

> 模块说明: 管理库存的实时查询、盘点、调拨、预警等核心功能
> 优先级: P0 (MVP必需)

---

## 一、库存查询

### Story 4.1 实时库存查询
**作为** 仓库管理员  
**我希望** 能够实时查询库存情况  
**以便** 了解当前库存数量和分布

**验收标准:**
- [ ] 可以按SKU、商品名称、条码查询库存
- [ ] 可以按仓库、库区、库位筛选
- [ ] 列表显示：SKU、商品名称、仓库、库位、批次、数量
- [ ] 显示可用数量、锁定数量、总数量
- [ ] 支持查看商品在各库位的分布
- [ ] 支持导出库存数据

**库存数据:**
```
总数量 = 可用数量 + 锁定数量
可用数量：可分配的库存
锁定数量：已分配未出库的库存
```

---

### Story 4.2 库存明细查询
**作为** 仓库管理员  
**我希望** 能够查看单个商品的详细库存分布  
**以便** 了解该商品在各库位的存储情况

**验收标准:**
- [ ] 点击商品可以查看库存明细
- [ ] 显示该商品在所有仓库的分布
- [ ] 显示每个库位的数量、批次、效期
- [ ] 显示该商品的入库记录和出库记录
- [ ] 显示该商品的库存变动历史

---

### Story 4.3 批次库存查询
**作为** 仓库管理员  
**我希望** 能够按批次查询库存  
**以便** 进行效期管理和先进先出

**验收标准:**
- [ ] 可以按批次号查询库存
- [ ] 显示批次对应的生产日期、过期日期
- [ ] 按过期日期排序（先过期在前）
- [ ] 临期商品标红显示
- [ ] 过期商品标灰显示并提示处理

## 二、库存盘点

### Story 4.4 盘点单创建
**作为** 仓库管理员  
**我希望** 能够创建盘点任务  
**以便** 定期核对实物与系统库存

**验收标准:**
- [ ] 可以创建全盘（整个仓库）
- [ ] 可以创建抽盘（指定库区/库位/商品）
- [ ] 可以按条件筛选创建盘点：
  - 按库区/库位范围
  - 按商品分类
  - 按ABC分类
  - 按动销频率
- [ ] 可以设置盘点类型：明盘（显示系统数量）/盲盘（不显示）
- [ ] 盘点单创建后锁定相关库位（禁止出入库）

---

### Story 4.5 盘点作业
**作为** 仓库盘点员  
**我希望** 能够进行盘点作业  
**以便** 记录实际库存数量

**验收标准:**
- [ ] 扫描库位码定位盘点位置
- [ ] 扫描商品条码确认商品
- [ ] 录入实际盘点数量
- [ ] 明盘模式下显示系统数量，差异自动计算
- [ ] 盲盘模式下不显示系统数量
- [ ] 可以拍照记录盘点情况
- [ ] 支持多次盘点（初盘/复盘）
- [ ] 盘点过程中发现差异可以标记

---

### Story 4.6 盘点差异处理
**作为** 仓库主管  
**我希望** 能够处理盘点差异  
**以便** 调整系统库存与实物一致

**验收标准:**
- [ ] 盘点完成后生成差异报告
- [ ] 差异报告显示：SKU、库位、系统数量、盘点数量、差异数量
- [ ] 可以逐条确认差异原因
- [ ] 差异原因可选：盘盈/盘亏/错放/漏扫/其他
- [ ] 可以批量确认生成库存调整单
- [ ] 差异调整需要审批（超过阈值）
- [ ] 审批通过后自动调整库存

---

### Story 4.7 盘点历史查询
**作为** 仓库管理员  
**我希望** 能够查询历史盘点记录  
**以便** 分析盘点准确率和问题

**验收标准:**
- [ ] 支持按时间范围、仓库、盘点类型查询
- [ ] 显示盘点单基本信息和状态
- [ ] 可以查看盘点明细和差异报告
- [ ] 统计盘点准确率、差异率
- [ ] 支持导出盘点报告

## 四、库存预警

### Story 4.12 库存预警设置
**作为** 仓库管理员  
**我希望** 能够设置库存预警规则  
**以便** 及时发现库存异常

**验收标准:**
- [ ] 可以设置安全库存预警（低于安全库存提醒）
- [ ] 可以设置超储预警（超过最大库存提醒）
- [ ] 可以设置效期预警（临期商品提醒）
- [ ] 可以设置呆滞预警（长期无动销提醒）
- [ ] 预警规则可以按SKU/分类/仓库设置

---

### Story 4.13 库存预警通知
**作为** 仓库管理员  
**我希望** 能够接收库存预警通知  
**以便** 及时处理库存异常

**验收标准:**
- [ ] 系统每日自动检查预警规则
- [ ] 触发预警时发送通知（站内消息/邮件）
- [ ] 预警列表显示所有当前预警
- [ ] 预警按紧急程度分级：紧急/重要/一般
- [ ] 可以标记预警已处理
- [ ] 可以设置预警免打扰时间

**预警类型:**
| 预警类型 | 触发条件 | 级别 |
|---------|---------|------|
| 库存不足 | 可用数量 < 安全库存 | 紧急 |
| 库存超储 | 可用数量 > 最大库存 | 重要 |
| 效期预警 | 距过期 ≤ 预警天数 | 重要 |
| 临期紧急 | 距过期 ≤ 预警天数/2 | 紧急 |
| 已过期 | 已过过期日期 | 紧急 |
| 呆滞库存 | 无出库 > 呆滞天数 | 一般 |

## 七、效期管理

### Story 4.17 效期预警监控
**作为** 仓库管理员  
**我希望** 能够监控商品的效期状态  
**以便** 及时处理临期和过期商品

**验收标准:**
- [ ] 系统自动计算商品效期状态
- [ ] 效期状态：正常/效期预警/临期/已过期
- [ ] 每日自动更新效期状态
- [ ] 效期预警商品列表显示
- [ ] 可以按效期状态筛选商品
- [ ] 支持导出效期预警报表

**效期状态规则:**
```
距过期 > 预警天数：正常
距过期 ≤ 预警天数：效期预警
距过期 ≤ 预警天数/2：临期
已过过期日期：已过期
```

---

### Story 4.18 FEFO出库策略
**作为** 系统  
**我希望** 出库时自动按FEFO（先过期先出）分配  
**以便** 减少过期损耗

**验收标准:**
- [ ] 出库分配时优先分配先过期的批次
- [ ] 同批次按入库时间先进先出
- [ ] 效期预警商品优先分配
- [ ] 可以手动指定批次出库（特殊情况）
- [ ] FEFO策略可以按商品设置开启/关闭
