# 日志审计 - 用户故事

> 模块说明: 操作日志、登录日志
> 优先级: P0 (MVP必需)

---

## Story 3.1 操作日志
**作为** 系统管理员
**我希望** 能够查看用户操作日志
**以便** 审计用户行为

**验收标准:**
- [ ] 记录所有增删改操作
- [ ] 显示操作人、操作时间、操作类型、操作对象
- [ ] 显示操作前/后的数据变化
- [ ] 支持按时间范围、用户、模块筛选
- [ ] 支持导出日志
- [ ] 日志保留90天

---

## Story 3.2 登录日志
**作为** 系统管理员
**我希望** 能够查看用户登录日志
**以便** 监控账号安全

**验收标准:**
- [ ] 记录用户登录/登出
- [ ] 显示登录时间、IP地址、登录方式
- [ ] 记录登录失败次数
- [ ] 支持按时间范围、用户筛选
- [ ] 异常登录（连续失败）标红提醒
