## 五、核心设计原则实现

### 数据关联原则 ✅
- 收货必须关联入库单
- 验收必须关联已收货的入库明细
- 上架必须关联已验收合格的入库明细
- 退货必须关联入库明细

### 单据编号规则 ✅
- 入库单号: IN + 年月日 + 4位序号
- 退货单号: RT + 年月日 + 4位序号
- 库存事务号: TX + 年月日 + 4位序号
- 批次号: 入库单号 + SKU后4位 + 2位序号
